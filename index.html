The Imitation Game
  • Un film de Morten Tyldum sorti en 2015
  • L'histoire du mathématicien Alan Turing, qui réussit à décrypter le code Énigma allemand
  • Quelle est la place de la femme dans cette mission ?


    • Articles



    Présentation

    Un film réalisé par Morten Tyldum, sorti en 2015.
    Il nous raconte comment Alan Turing décrypta le code Énigma allemand.
    Il nous montre également la place des femmes dans ce milieu.

    Résumé

    Découvrez le résumé complet du film : contexte, personnages et intrigue principale.

    Personnages

    Nous nous intéresserons particulièrement à la place de la femme dans ce film, en analysant principalement le personnage de Joan Clarke.

    Analyse

    Nous avons analysé la place de Joan Clarke, une femme cryptologue, dans le milieu naissant de l'informatique des années 40. Nous avons relevé les éventuelles formes de discrimination (mise à l’écart, rabaissement, écarts de paiement…).



    Description

    Site web créé dans le cadre du projet tutoré. Il traite du film The Imitation Game et se rapporte à la place des femmes dans le monde de l'informatique dans les années 1940.
